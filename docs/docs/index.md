# Code Issues Solver

Autonomous web app testing, detection, and fixing system.

## Features

- Playwright-based automated testing
- AI-powered bug detection and patching
- GitHub integration (issues, PRs, webhooks)
- Multi-provider LLM support (OpenRouter, Alibaba Cloud)
- Real-time log streaming via SSE
- Full CRUD UI with React 18
